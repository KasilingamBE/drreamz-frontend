# Parkyourself Frontend Monorepo 
# Next.js + React Native
Install Dependencies:
```sh
$ yarn install
```

# Next.js Web
Run Web Next.js:
```sh
$ yarn web
```

Build Web:
```sh
$ yarn web:build
```
Deploy Web:
```sh
$ yarn web:deploy
```


# React Native
Start metro server:
```sh
$ yarn mobile
```
Run android:
```sh
$ yarn android
```
Run ios:
```sh
$ yarn ios
```
